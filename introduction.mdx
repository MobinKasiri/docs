## How It Works

<Steps>
  <Step title="Create Account & API Key">
    Sign up at [nexoranode.xyz](https://nexoranode.xyz) and generate your first API key from the [dashboard](https://app.nexoranode.xyz). Your key authenticates every RPC and management API request you make.
  </Step>
  <Step title="Choose a Network and Provision Endpoint">
    Select from 20+ supported networks — including Ethereum, Polygon, Arbitrum, Solana, and more. NexoraNode instantly provisions a dedicated endpoint URL scoped to your API key and chosen network.
  </Step>
  <Step title="Start Making RPC Calls">
    Point any HTTP client, ethers.js, viem, web3.py, or other web3 library at your endpoint and start querying the chain. No additional setup required.
  </Step>
</Steps>
